# ☕ Tizi-Café

## 📌 Description du site

**Tizi-Café** est un site vitrine pour un coffee shop.  
Il permet de consulter un menu dynamique avec filtrage par catégories (cafés et pâtisseries), de simuler un système de commande, ainsi que de gérer une inscription et une connexion simulées côté client avec validation des formulaires.

Les données utilisateur sont stockées localement dans le **LocalStorage** du navigateur.

---

## 🚀 Fonctionnalités

- Affichage dynamique du menu
- Filtrage des produits par catégories :
- Cafés
- Pâtisseries
- Simulation de prise de commande
- Système d’inscription (front-end uniquement)
- Système de connexion (front-end uniquement)
- Stockage des comptes via LocalStorage
- Validation des formulaires côté client

---

## ⚙️ Installation et lancement

### 1. Installer le serveur statique

Ouvrez un terminal dans le dossier du projet et exécutez :

```bash
npm install serve
```

### 2. Lancer le serveur

```bash
npx serve
```

### 3. Accéder au site

Une fois le serveur lancé, ouvrez l’URL indiquée dans le terminal (généralement) :

```
http://localhost:3000
```

## 🧪 Utilisation

1. Aller dans "Nos Produits" pour tester le filtrage dynamique du menu.
2. Aller dans "S’inscrire" pour créer un compte (stocké dans le navigateur).
3. Aller dans "Se connecter" pour tester l’authentification avec le compte créé.
4. Simuler une commande depuis les produits.
